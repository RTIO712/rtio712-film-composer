## 업로드 방식
1. Upload Studio에서 장르를 선택합니다.
2. 트랙 제목을 입력합니다.
3. 오디오 파일을 선택합니다.
4. "파일 업로드 추가"를 누릅니다.
5. 해당 장르 카테고리 카드에 가장 최근 업로드 곡이 자동 표시됩니다.
6. 장르 카드를 클릭하면 전체 트랙 라이브러리를 확인할 수 있습니다.

## 저장 방식
- 브라우저 LocalStorage에 저장됩니다.
- 같은 브라우저 / 같은 기기에서 유지됩니다.
- 다른 기기와 자동 동기화되지는 않습니다.

## 서버 업로드 방법
1. ZIP을 압축 해제합니다.
2. index.html, style.css, script.js, assets 폴더를 서버 루트(public_html / htdocs / www)에 업로드합니다.
3. index.html이 최상위에 있어야 합니다.

## GitHub Pages 업로드 방법
1. GitHub 저장소를 엽니다.
2. index.html, style.css, script.js, README_DEPLOY_KR.md, assets 폴더를 업로드합니다.
3. Settings > Pages 에서 main / root 배포를 사용합니다.
