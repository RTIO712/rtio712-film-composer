OVERVIEW

화이트 중심의 럭셔리 블랙 & 화이트 인터페이스

왼쪽 정렬과 중앙 정렬을 적절히 섞고, 라인과 분할선을 사용해 전체 구조를 보다 정돈되고 세련되게 보이도록 구성했습니다.

01

40% 이상 화이트 사용

배경, 카드, 콘텐츠 영역에서 화이트 비율을 높여 더 밝고 고급스럽게 보이도록 개선했습니다.

02

상단 업로드 전용 패널

업로드는 상단 전용 창에서 진행하고, 메인 장르 화면에서는 업로드된 파일들을 배너 형태로 확인하도록 분리했습니다.

03

장르별 파일 배너

장르를 클릭하면 해당 장르에 업로드된 파일 목록이 배너 형태로 열리고, 제목과 파일명, 오디오 재생을 확인할 수 있습니다.

04

클린하고 시크한 분할선

구간별 라인, 카드 테두리, 섹션 분리선을 사용해 웹사이트 전체가 더 정갈하고 전문적인 인상을 가지도록 구성했습니다.
